# Soft-Mart

## Project Documentation

### Description
Soft-Mart is an online platform enabling users to buy and sell a variety of items efficiently.

### Features
- User-friendly interface
- Secure transactions
- Multi-category listings
- Search and filter options
- Review and rating system

### Installation
To install the project, follow these steps:
1. Clone the repository: `git clone https://github.com/samadhi-hansika/Soft-Mart.git`
2. Navigate to the project directory: `cd Soft-Mart`
3. Install required dependencies: `npm install`

### Usage
After installation, run the application using:
`npm start`
